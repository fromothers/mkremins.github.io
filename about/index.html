I'm Max Kreminski – an interaction designer, computer programmer, writer, and undergraduate student at USC's Interactive Media and Games Division. I write code (mostly Clojure) and share a lot of what I write on GitHub.

I enjoy reading, thinking, and occasionally writing about creativity-enabling tools, especially software tools for authoring interactive experiences. I'm particularly interested in human-friendly environments for computer programming, whatever forms they might take.

Major ongoing projects include:

  • Tailspin – an experimental reactive notebook-style Clojure REPL
  • Flense – an experimental structural editor for Clojure code
  • Stilts – a tiny Clojure interpreter, written in ClojureScript
  • Fanciful – an expressive text formatting library for Bukkit plugins

You should follow me on Twitter. You can also reach me via email (maxkreminski at Gmail) if you're into that sort of thing.
